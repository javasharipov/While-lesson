What is a While Loop?

A while loop is a control flow statement that allows code to be executed repeatedly based on a given boolean condition.

Syntax

while (condition) {
    // code block
}

Examples

Here is an example of a while loop:

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
